# fairness-ml-project
Machine Learning model with fairness and explainability analysis
## overview
this project builds a logistic regression model using the adult the adult income dataset and evaluates fairness using fairlean. it also applies shap and lime for explainability.
## files included
- faieness_project.ipynb main notbook
- notebook_output.pdf notebook with outputs
- report.pdf 3-page report
## tools used
- python
- scikit-learn
- fairlearn
- shap
- lime
  ## how to run
  - open the notebook in google in google colab
  - run all cells
  - install required libraries if prompted

    ## results
    - accuracy: ~80-85%
    - fairness dofferences observed across gender groups
    - key features: education, age, hours-per-week
